# Skills: 一天重塑人生

如何在一天内重塑你的人生，6 个可执行 skills，帮助你真正改变人生而非两周放弃。

## 快速开始

### 你有哪些 skill？

| Skill | 用途 |
|-------|------|
| [identity-level](skills/identity-level.md) | 从"行动层面"切换到"身份层面"，解决总是反弹的问题 |
| [anti-vision](skills/anti-vision.md) | 用"害怕失去"驱动行动，解决不知道想要什么的问题 |
| [mind-stage](skills/mind-stage.md) | 定位心智发展阶段，解决迷茫"我是谁"的问题 |
| [cybernetics](skills/cybernetics.md) | 建立反馈迭代系统，解决总是做不好的问题 |
| [one-day-reset](skills/one-day-reset.md) | 用一天做深度复盘，重新规划人生方向 |
| [gamification](skills/gamification.md) | 把人生变成游戏，解决坚持不下来问题 |
| [refixlife](skills/refixlife.md) | 智能技能路由器，根据用户描述的问题状态自动匹配并触发对应的反思行动技能 |

### 快速判断该用哪个

```
不知道想要什么 → anti-vision
努力但没效果 → identity-level + cybernetics
想彻底改变 → one-day-reset
坚持太苦 → gamification
迷茫我是谁 → mind-stage
```

## 使用示例

### 示例 1：减肥总是反弹

```
用户: "我想减肥，但每次都坚持不下来"

分析: "坚持不下来" → identity-level

1. 问: 如果要减肥成功，你应该成为什么样的人?
    → "我是那种吃得健康的人"

2. 问: 如果已经是这种人，这周你会自然做什么?
    → "看到垃圾食品就不想吃" (不是强迫自己)

3. 建立身份保护:
    → 当你想吃垃圾食品时，你不是在"坚持"或"放弃"
      而是在选择"你是谁"
```

### 示例 2：学英语半途而废

```
用户: "我想学英语，但每次都半途而废"

1. 诊断: 只关注"学英语"(行动)，没想过"成为哪种人"(身份)

2. 建立身份:
    → "我是那种能用英语思考的人" (不是"我要背单词")

3. 构建反愿景:
    → 5年后英语最差: 出国不敢开口、错过好机会
    → 一句话: "我绝不允许自己变成不敢开口的人"

4. 反馈系统:
    → 目标: 能开口说英语
    → 行动: 每天练习30分钟
    → 感知: 录音对比、找语伴反馈
    → 迭代: 调整方法

5. 游戏化:
    → 一年目标: 日常交流 (主线任务)
    → 一月项目: 日常口语100句 (升级打怪)
    → 每日行动: 晨读30分钟 (日常任务)
```

详细示例见 [examples/](examples/)

## 核心方法论

### 身份层级模型

行为改变存在两个层面：

- **第二层面 (行动)**：强迫自己做事 ("今年要更自律")。短期有效，很快反弹。
- **第一层面 (身份)**：先改变"我是谁"。身份改变自然带来行动改变。

### 反愿景构建

人对"不想要什么"的判断比"想要什么"更清晰。用负能量驱动：

1. 描述 5 年后最糟糕的星期二
2. 描述 10 年后错过的机会
3. 临终视角的代价
4. 一句话反愿景压缩版

### 控制论反馈系统

智能系统的反馈循环：

```
目标 → 行动 → 感知 → 比较 → 迭代
```

高智力 = 能从这个循环中学习并坚持。
低智力 = 卡在问题上，遇到阻碍就放弃。
