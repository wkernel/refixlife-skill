# 心智层级发展模型

## R — 原文 (Reading)

> 自我发展九阶段：冲动阶段（冲动就是行动，没有区分）→自我保护阶段（世界是危险的，你必须学会保护自己）→从众阶段（你=你的群体，群体的规则就像绝对真理）→自我觉察阶段（你开始意识到"内在世界"和"外在表现"并不总是一致）→尽责阶段（你建立属于自己的原则体系，并对自己负责）→个体化阶段（你开始意识到你的"原则"也受环境塑造，开始不再那么执着）→战略家阶段（你能与系统共舞，同时意识到自己也是系统的一部分）→建构觉察阶段（你看到所有框架都只是"有用的虚构"）→合一阶段（自我与生命的分离消失）
>
> — 作者, 第4节

---

## I — 方法论骨架 (Interpretation)

人的心智会随着时间按照可预测的模式进化,分九个阶段:

**早期阶段(3-4之前):**
- 冲动阶段: 行动=情绪,没有区分对错
- 自我保护阶段: 世界是危险的,学会保护自己
- 从众阶段: 群体规则=绝对真理

**中期阶段(4-7):**
- 自我觉察阶段: 意识到内外不一致,开始质疑
- 尽责阶段: 建立个人原则体系和责任
- 个体化阶段: 意识到原则也受环境影响,不再执着

**高级阶段(7-9):**
- 战略家阶段: 能与系统共舞,意识到自己也是系统一部分
- 建构觉察阶段: 看到框架只是"有用的虚构"
- 合一阶段: 自我与生命分离消失

大多数人在4-8之间。穿越每个阶段遵循相同模式。关键是不让心智在某阶段"固化"。

## A1 — 书中的应用 (Past Application)

### 案例: 不同阶段的"看问题"

- **从众阶段**: 完全无法理解为什么会有人投与自己家庭不同的票
- **自我觉察阶段**: 坐在教堂里发现自己并不确定是否相信周围人相信的东西
- **战略家阶段**: 参与政治时知道自己的视角是局部的,有各种看不见的偏见

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 用户问"我是谁"、"人生意义是什么"
2. 用户说"我感到迷茫,不知道想要什么"
3. 用户在经历重大人生转变(辞职/离婚/出柜等)
4. 用户说"我和父母/朋友的想法不一样,我是不是有问题"

### 语言信号 (用户的话里出现这些就应激活)

- "我是谁"
- "人生意义"
- "迷茫"
- "他们都不能理解我"
- "我和别人不一样"
- "为什么活着"

### 与相邻 skill 的区分

- 与 `身份层级` 的区别: 身份层级是"你是谁",心智层级是"你处于什么发展阶段"
- 与 `一天流程` 的区别: 心智层级是定位,一天流程是行动

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **帮助用户定位当前阶段**
   - 问:"面对以下情况你会怎么反应?"
   - 提供各阶段特征让用户自我对照
   - 完成标准: 用户能识别自己在哪个阶段

2. **识别固化风险**
   - 问:"你最近有拒绝什么新观点吗?""你觉得'这就是我'了吗?"
   - 完成标准: 用户能识别是否有固化倾向

3. **提供阶段穿越路径**
   - 告诉用户: 穿越每个阶段的方式相同
   - 提供通用的"挑战-反思-整合"模式
   - 完成标准: 用户理解下一阶段的进入方式

4. **警惕极端情况**
   - 判停: 若用户处于1-2阶段或表达轻生念头,建议寻求专业帮助
   - 完成标准: 及时转介

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 用户只是问实用技能,不涉及自我认知
- 用户处于危机状态(需要专业帮助)
- 用户不想自我探索,只想解决问题

### 作者在书中警告的失败模式

- **心智固化**: "如果不小心,你的心智会在某个阶段'固化',让你之后很难过上真正有意义的生活"

### 作者的盲点 / 时代局限

- 假设读者心智在4-8阶段
- 未考虑更早期阶段读者的实际困境
- 模型整合自多个理论,未经严格验证

### 容易混淆的邻近方法论

- 马斯洛需求层次: 生理→安全→social→尊重→自我实现,不同模型
- 螺旋动力学: 价值观演化的颜色模型,更学术化
