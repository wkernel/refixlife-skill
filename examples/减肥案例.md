# 案例：减肥总是反弹

## 问题描述
用户: "我想减肥，但每次都坚持不下来，减了又胖回来"

## 诊断过程
关键词: "坚持不下来" → 触发 identity-level skill

## 解决步骤

### 第一步：确认行动层面 vs 身份层面
**教练问**: 你现在在说"要做什么"还是"要成为谁"?
**用户答**: 我想减肥20斤（行动层面）

### 第二步：引导身份转换
**教练问**: 如果要达成这个目标，你应该成为什么样的人?
**用户答**: 我应该是那种自律的人

**教练引导**: 更具体地说，你是什么样的饮食习惯的人?
**用户答**: 我是那种吃得健康的人（身份陈述）

### 第三步：验证身份自然行为
**教练问**: 如果已经是"那种吃得健康的人"，你这周会自然做什么?
**用户答**: 
- 看到蛋糕不用纠结就不吃
- 外食时自然选蔬菜多的
- 超市不买垃圾食品回家

**结论**: 这些行为是自然的，不需要意志力硬撑 ✓

### 第四步：建立身份保护机制
**教练提醒**: 当你想吃垃圾食品时，你不是在考虑"我应该坚持吗"，而是在问"我是不是那种吃得健康的人?"

### 第五步：结合其他 skill（可选）
- 如果还缺乏动力 → 用 anti-vision 想象5年后继续发胖的样子
- 如果需要反馈 → 用 cybernetics 建立每周体重测量+饮食记录
- 如果觉得太枯燥 → 用 gamification 设定每周达标奖励

## 结果
用户报告：第二周开始不用靠意志力，饮食选择变得自然，一个月减了8斤且没有反弹倾向。

## 关键洞察
不是你"缺乏意志力"，而是你在错误的层面（行动）上努力。真正的改变发生在身份层面：先成为那种人，行为自然跟随。
