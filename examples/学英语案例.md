# 案例：学英语半途而废

## 问题描述
用户: "我想学英语，但每次都半途而废，买了很多课程没坚持完"

## 诊断过程
关键词: "半途而废" → 可能涉及多个 skill
- 先检查身份: 用户只在说"学英语"(行动)而没说"成为哪种人"
- 再检查动力: 可能缺乏反愿景驱动
- 再检查反馈: 可能不知道自己进步如何
- 再检查趣味性: 可能觉得太枯燥

## 解决步骤

### 第一步：建立身份层面
**教练问**: 你为什么想学英语?
**用户答**: 为了能看懂美剧、为了工作升职、为了出国旅游不尴尬

**教练引导**: 如果你英语很好，你应该成为什么样的人?
**用户答**: 我应该是那种能用英语思考的人
（身份陈述：我是那种能用英语思考的人）

### 第二步：构建反愿景（增加动力）
**教练问**: 5年后如果英语还是现在这个水平，最糟糕的情况是什么?
**用户答**: 
- 出国旅游完全靠导游，不敢自己点餐
- 工作会议全程听不懂，只能点头
- 错过很多国际机会，看着同事升职

**教练问**: 用一句话概括你绝不允许自己变成什么样?
**用户答**: 我绝不允许自己变成在重要场合哑巴的人

### 第三步：建立反馈系统（知道自己在进步）
**教练问**: 你现在怎么知道自己的英语进步了？
**用户答**: 我不知道，只靠感觉

**教练建议**: 
- 目标: 能不带字幕看一部喜剧的第一集
- 行动: 每天听力+模仿30分钟
- 感知: 每周对照原文看自己能理解多少%
- 比较: 每周记录理解率，目标从30%升到80%
- 迭代: 如果某方法无效，换资源或方法

### 第四步：游戏化（增加趣味性和坚持性）
**教练问**: 怎样让这个过程像玩游戏一样上瘾？
**用户答**: 设定级别和奖励

**一起设计**:
- 一年目标（主线任务）: 能流畅看美剧不带字幕 → 解锁"美剧达人"称号
- 一月项目（升级打怪）: 本月掌握日常场景100句 → 每完成20句升一级
- 每日行动（日常任务）: 晨练30分钟 → 每天完成得100经验值
- 底线约束（游戏规则）: 即便忙也要保证5分钟不中断

### 第五步：执行与复盘
**用户执行反馈**:
- 第一周: 身份认同建立完成，开始每天晨练
- 第二周: 通过录音对比发现发音问题，调整方法（跟读+纠正）
- 第三周: 达成第一个微目标：能理解简单对话的50%
- 第四周: 坚持度达到90%，开始感到语感

## 结果
用户报告：两个月后能看懂简单美剧，三个月后能进行基本交流，六个月后工作英文汇报无压力。

## 关键洞察
学英语不是"背单词表"（行动），而是要成为"能用英语思考的人"（身份）。
动力来自害怕失去机会（反愿景），坚持靠游戏化和及时反馈（反馈系统+游戏化）。
